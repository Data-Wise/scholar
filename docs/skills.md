## Usage Examples

### Automatic Activation

Skills activate based on conversation keywords:

```
User: "How do I prove identifiability of mediation effects?"
→ Activates: identification-theory, mathematical-foundations
→ Provides: Identifiability conditions, proof strategy

User: "Design a simulation to compare bootstrap methods"
→ Activates: simulation-architect, computational-inference
→ Provides: Simulation parameters, performance metrics

User: "Write methods section for my paper"
→ Activates: methods-paper-writer, methods-communicator
→ Provides: Statistical notation, clear exposition
```

### Skill Coordination

Multiple skills work together:

```
Task: "Develop new mediation method"

Active skills:
1. mathematical-foundations → Theory development
2. identification-theory → Identifiability proof
3. algorithm-designer → Estimation algorithm
4. simulation-architect → Performance evaluation
5. methods-paper-writer → Manuscript preparation
```
